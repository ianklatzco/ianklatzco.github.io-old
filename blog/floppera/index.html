Floppera

May 27th, 2015 4:28 am

Ever wanted to hear floppy drives sing?

I stole some floppy drives from old computers in the basement and started following Sammy1Am’s tutorial. The system is basically an Arduino, 5v power supply for the Arduino/floppies, a Java program, some Arduino code, and as many floppies as you want. I’m not going to go in to how you set everything up, as that’s been done plenty of times already. It’s certainly been a lot of fun to play various music files in floppified version, though!
